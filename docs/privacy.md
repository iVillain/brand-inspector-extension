# Політика конфіденційності Brand Inspector

Останнє оновлення: 2025

Розширення **Brand Inspector** обробляє вміст *лише активної вкладки* виключно після натискання кнопки **Scan**. Усі обчислення виконуються *локально в браузері користувача*. Розширення **не збирає**, **не зберігає** та **не передає** персональні дані третім сторонам.

## Які дані обробляються
- HTML та CSS активної сторінки  
- Логотипи, SVG-графіка  
- Шрифти та їх джерела  
- Кольорова палітра  
- Видимий текст сторінки (для формування ключових слів)

Усі дані обробляються **в памʼяті браузера** та **не виходять за його межі**.

## Жодних сторонніх передач даних
Розширення:
- не використовує аналітики  
- не має бекенду  
- не зберігає історію  
- не передає інформацію за замовчуванням  
- не використовує cookies  
- не взаємодіє з третіми сервісами

## Зовнішні API (опційно)
Користувач може ввести **власний API ключ**, якщо хоче покращити фільтрацію зображень через сторонній AI.  
Це **повністю добровільно**, і виклики робляться **лише на вказаний користувачем endpoint**.
