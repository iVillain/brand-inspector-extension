# Brand Inspector — Chrome Extension

Brand Inspector — це розширення Chrome, яке сканує активну вкладку та витягує:

- логотипи та SVG
- ключові слова
- короткий опис
- шрифти з джерелами (Google Fonts, Adobe, Custom)
- кольорову палітру сторінки  
- інші бренд-елементи

Усі обчислення виконуються **локально в браузері**. Розширення **не передає жодних даних** третім сторонам.

## Функції (MVP)
- Аналіз DOM/CSS на поточній сторінці
- Евристичний пошук логотипів
- Пошук шрифтів + визначення джерела
- Генерація кольорової палітри
- Генерація keywords з фільтром stopwords.json
- Експорт JSON / Copy to Figma (WIP)

## Інсталяція в режимі розробника
1. Завантаж ZIP або клонуй репозиторій.
2. В Chrome → `chrome://extensions/`.
3. Увімкни **Developer mode**.
4. Натисни **Load unpacked**.
5. Обери папку розширення.
